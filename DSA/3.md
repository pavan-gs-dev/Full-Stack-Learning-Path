# Linked List in Python - Complete Beginner Guide

# What is a Linked List?

A Linked List is a linear data structure where elements are connected using pointers (references).

Unlike arrays, linked list elements are NOT stored in continuous memory locations.

Each element is called a **Node**.

Each node contains:

1. Data
2. Address (Reference) of the next node

---

# Real Life Example

Think of a treasure hunt.

```text
Paper 1 -> "Go to Room 2"
Paper 2 -> "Go to Room 5"
Paper 3 -> "Treasure Here"
```

Each paper tells you where the next paper is.

This is exactly how a Linked List works.

---

# Array vs Linked List

## Array

```text
Index:  0   1   2   3
Value: 10  20  30  40
```

Stored continuously in memory.

Advantages:

* Fast access
* Easy indexing

Disadvantages:

* Insertion is slow
* Deletion is slow

---

## Linked List

```text
10 -> 20 -> 30 -> 40 -> NULL
```

Nodes can be anywhere in memory.

Advantages:

* Easy insertion
* Easy deletion

Disadvantages:

* Slow searching
* No indexing

---

# Structure of a Node

A node contains:

```text
+---------+---------+
|  Data   |  Next   |
+---------+---------+
```

Example:

```text
+-----+------+     +-----+------+     +-----+------+
| 10  |  *---|---->| 20  |  *---|---->| 30  | NULL |
+-----+------+     +-----+------+     +-----+------+
```

---

# Creating a Node in Python

```python
class Node:

    def __init__(self, data):

        self.data = data

        self.next = None
```

---

# Understanding the Code

```python
node = Node(10)
```

Memory:

```text
Data = 10

Next = None
```

Visualization:

```text
+-----+------+
```


---

# Connecting Nodes

```python

```


---

# Full Example

```python
class Node:

    def __init__(self, data):

        self.data = data

        self.next = None


first = Node(10)
second = Node(20)
third = Node(30)

first.next = second
second.next = third

print(first.data)
print(first.next.data)
print(first.next.next.data)
```

Output:

```text
10
20
30
```

---

# Traversing a Linked List

Traversing means visiting every node.

```python
current = first

while current:

    print(current.data)

    current = current.next
```

Output:

```text
10
20
30
```

# Insert at Beginning

Before:

```text
10 -> 20 -> 30
```

Insert:

```text
5
```

After:

```text
5 -> 10 -> 20 -> 30
```

Code:

```python
new_node = Node(5)

new_node.next = first

first = new_node
```

Time Complexity:

```text
O(1)
```

Very Fast

---

# Insert at End

Before:

```text
10 -> 20 -> 30
```

After:

```text
10 -> 20 -> 30 -> 40
```

Code:

```python
new_node = Node(40)

current = first

while current.next:
    current = current.next

current.next = new_node
```

Time Complexity:

```text
O(n)
```

---

# Search Element

```python
target = 20

current = first

while current:

    if current.data == target:
        print("Found")

    current = current.next
```

Time Complexity:

```text
O(n)
```

---

# Reverse Linked List

Before:

```text
10 -> 20 -> 30 -> NULL
```

After:

```text
30 -> 20 -> 10 -> NULL
```

Code:

```python
prev = None

current = first

while current:

    next_node = current.next

    current.next = prev

    prev = current

    current = next_node

first = prev
```

Time Complexity:

```text
O(n)
```

---

# Types of Linked Lists

## 1. Singly Linked List

```text
10 -> 20 -> 30 -> NULL
```

Each node points only to next node.

---

## 2. Doubly Linked List

```text
NULL <- 10 <-> 20 <-> 30 -> NULL
```

Each node stores:

* Previous Node
* Next Node

---

## 3. Circular Linked List

```text
10 -> 20 -> 30
^           |
|___________|
```

Last node points to first node.
