# ARRAY (Most Important Topic)

## What is an Array?

Array is a collection of similar elements stored in contiguous memory locations.

Example:

```text
Index:  0   1   2   3   4
Value: 10  20  30  40  50
```

In Python, we usually use Lists as Arrays.

```python
arr = [10, 20, 30, 40, 50]
```

---

# Array Characteristics

### 1. Fixed Position Index

Every element has an index.

```python
arr = [10,20,30]

print(arr[0])
```

Output:

```text
10
```

---

### 2. Index Starts from 0

```python
arr = [100,200,300]

print(arr[0])
print(arr[1])
print(arr[2])
```

Output:

```text
100
200
300
```

---

# Array Operations

## Access Element

```python
arr = [10,20,30,40]

print(arr[2])
```

Output:

```text
30
```

Time Complexity:

```text
O(1)
```

Very Fast

---

## Insert Element

```python
arr = [10,20,40]

arr.insert(2,30)

print(arr)
```

Output:

```text
[10,20,30,40]
```

Time Complexity:

```text
O(n)
```

---

## Append Element

```python
arr = [10,20,30]

arr.append(40)

print(arr)
```

Output:

```text
[10,20,30,40]
```

---

## Delete Element

```python
arr = [10,20,30,40]

arr.remove(30)

print(arr)
```

Output:

```text
[10,20,40]
```

Time Complexity:

```text
O(n)
```

---

## Update Element

```python
arr = [10,20,30]

arr[1] = 200

print(arr)
```

Output:

```text
[10,200,30]
```

---

# Array Traversal

Visiting every element once.

```python
arr = [10,20,30,40,50]

for i in arr:
    print(i)
```

Output:

```text
10
20
30
40
50
```

Time Complexity:

```text
O(n)
```

---

# Array Searching

## Linear Search

Search one by one.

```python
arr = [10,20,30,40,50]

target = 40

for i in range(len(arr)):
    if arr[i] == target:
        print("Found at index", i)
```

Output:

```text
Found at index 3
```

Time Complexity:

```text
O(n)
```

---

# Array Reverse

Method 1

```python
arr = [10,20,30,40]

arr.reverse()

print(arr)
```

Output:

```text
[40,30,20,10]
```

Method 2

```python

print(arr[::-1])
```

# Array Slicing

```python
arr = [10,20,30,40,50]
```

Get first 3 elements

```python
print(arr[:3])
```

Output

```text
[10,20,30]
```

Get last 2 elements

```python
print(arr[-2:])
```

Output

```text
[40,50]
```
