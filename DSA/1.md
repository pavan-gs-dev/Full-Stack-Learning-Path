# Data Structures and Algorithms (DSA) using Python

# 📚 Introduction to DSA

DSA stands for **Data Structures and Algorithms**.

* **Data Structure** = A way of organizing and storing data efficiently.
* **Algorithm** = A step-by-step procedure to solve a problem.

Think of it like this:

| Real Life             | DSA            |
| --------------------- | -------------- |
| Bookshelf             | Data Structure |
| Method to find a book | Algorithm      |

Example:

Suppose you have 10,000 student records.

Without DSA:

* Searching becomes slow.
* Memory usage increases.

With DSA:

* Searching becomes fast.
* Memory usage is optimized.
* Programs become efficient.

# 1. Primitive Data Structures

Primitive means basic built-in data types.

Examples:

```python
a = 10        # Integer
b = 3.14      # Float
c = "Pavan"   # String
d = True      # Boolean
```

### Types

| Type      | Example |
| --------- | ------- |
| Integer   | 10      |
| Float     | 5.6     |
| Character | 'A'     |
| String    | "Hello" |
| Boolean   | True    |

---

# Linear Data Structures

Data is stored sequentially.

Example:

```text
10 → 20 → 30 → 40 → 50
```

Types:

* Array
* Linked List
* Stack
* Queue

---

# Non-Linear Data Structures

Data is not stored sequentially.

Examples:

Tree

```text
      10
     /  \
    20  30
```

Graph

```text
A ---- B
|      |
|      |
C ---- D
```
