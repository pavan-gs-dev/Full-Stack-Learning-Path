# Python DSA Interview Questions

## Question 1: Minimum Number of Swaps Required to Sort an Array

### Problem

Given an unsorted array, find the minimum number of swaps required to sort it.

### Example

```python
arr = [4, 1, 3, 2]
```

Sorted Array:

```python
[1, 2, 3, 4]
```

### Explanation

Swap 4 and 1:

```python
[1, 4, 3, 2]
```

Swap 4 and 2:

```python
[1, 2, 3, 4]
```

Total Swaps = 2

### Simple Idea

* Find where each element should be.
* If an element is not in its correct position, swap it.
* Count the swaps.

## Question 2: Find the Largest Element

### Problem

Find the largest element without using `max()`.

### Input

```python
arr = [10, 50, 20, 80, 30]
```

### Solution

```python
arr = [10, 50, 20, 80, 30]

largest = arr[0]

for i in range(len(arr)):
    if arr[i] > largest:
        largest = arr[i]

print(largest)
```

### Output

```python
80
```

### Explanation

Start by assuming the first element is the largest.

```python
largest = 10
```

Compare each element:

```text
50 > 10  -> largest = 50
20 > 50  -> No
80 > 50  -> largest = 80
30 > 80  -> No
```

Final Answer:

```python
80
```

## Question 3: Reverse an Array Without Using reverse()

### Problem

Reverse the array without using `reverse()`.

### Input

```python
arr = [10, 20, 30, 40, 50]
```

### Solution (Two Pointers)

```python
arr = [10, 20, 30, 40, 50]

left = 0
right = len(arr) - 1

while left < right:
    arr[left], arr[right] = arr[right], arr[left]

    left += 1
    right -= 1

print(arr)
```

### Output

```python
[50, 40, 30, 20, 10]
```

### Explanation

Initial:

```text
10 20 30 40 50
↑           ↑
L           R
```

Swap:

```text
50 20 30 40 10
```

Move pointers inward:

```text
50 20 30 40 10
   ↑     ↑
   L     R
```

Swap again:

```text
50 40 30 20 10
```

Done.

## Question 4: How to Use Swap

### Problem

Exchange the positions of two elements.

### Example

```python
a = 10
b = 20

a, b = b, a

print(a)
print(b)
```

### Output

```python
20
10
```

### Swap Elements in Array

```python
arr = [10, 20, 30]

arr[0], arr[2] = arr[2], arr[0]

print(arr)
```

### Output

```python
[30, 20, 10]
```

### Explanation

Before:

```text
Index: 0  1  2
Value:10 20 30
```

After Swap:

```text
Index: 0  1  2
Value:30 20 10
```

# Notes

## What is an Array/List?

```python
arr = [10, 20, 30, 40]
```

Python calls this a **List**.

In interviews, people often call it an **Array**.

---

## What is an Index?

```text
Index:  0   1   2   3
Value: [10, 20, 30, 40]
```

```python
arr[0] = 10
arr[1] = 20
arr[2] = 30
arr[3] = 40
```

---

## What is a Pointer?

A pointer is a variable storing an index position.

Example:

```python
left = 0
right = len(arr) - 1
```

```text
10 20 30 40 50
↑           ↑
L           R
```

Used in:

* Reverse Array
* Pair Sum
* Sorted Arrays
* DSA Interview Questions
