
# ARRAY (Most Important Topic)

## What is an Array?

Array is a collection of similar elements stored in contiguous memory locations.
OR An array is a collection of multiple values stored in a single variable.

Example:

```text
Index:  0   1   2   3   4
Value: 10  20  30  40  50
```

In Python, we usually use Lists as Arrays.

```python
arr = [10, 20, 30, 40, 50]
```

## What Happens in Memory?

Array:

```text
Index:  0   1   2   3   4
```

# 1. Rotate Array By K Positions

Question:

```text
Input:
[1,2,3,4,5]
K = 2

Output:
[4,5,1,2,3]
```

---

## Approach

Take last K elements and move them to front.

### Python

```python
arr = [1,2,3,4,5]

k = 2

k = k % len(arr)

result = arr[-k:] + arr[:-k]

print(result)
```

Output:

```text
[4,5,1,2,3]
```

---

## Dry Run

```text
arr = [1,2,3,4,5]

Last 2 elements:
[4,5]

Remaining:
[1,2,3]

Combine:
[4,5,1,2,3]
```

Time Complexity:

```text
O(n)
```

Space Complexity:

```text
O(n)
```

# 2. Move All Zeros To End

Question

```text
Input:
[1,0,2,0,3,0,4]

Output:
[1,2,3,4,0,0,0]
```

---

## Optimized Approach

Use two pointers.

### Python

```python
arr = [1,0,2,0,3,0,4]

position = 0

for i in range(len(arr)):
    if arr[i] != 0:
        arr[position], arr[i] = arr[i], arr[position]
        position += 1

print(arr)
```

Output

```text
[1,2,3,4,0,0,0]
```

Time Complexity:

```text
O(n)
```

Space Complexity:

```text
O(1)
```

# 5. Kadane's Algorithm

Most Famous Interview Question

Question:

Find maximum sum contiguous subarray.

Input

```text
[-2,1,-3,4,-1,2,1,-5,4]
```

Output

```text
6
```

Subarray

```text
[4,-1,2,1]
```

---

## Optimized Solution

Python

```python
arr = [-2,1,-3,4,-1,2,1,-5,4]

current_sum = arr[0]
max_sum = arr[0]

for i in arr[1:]:
    current_sum = max(i, current_sum + i)
    max_sum = max(max_sum, current_sum)

print(max_sum)
```

Output

```text
6
```

Time Complexity

```text
O(n)
```

Space Complexity

```text
O(1)
```

Interview Tip:

Amazon, Microsoft and Google frequently ask Kadane's Algorithm.

# 6. Merge Two Sorted Arrays

Input

```text
[1,3,5]
[2,4,6]
```

Output

```text
[1,2,3,4,5,6]
```

Python

```python
a = [1,3,5]
b = [2,4,6]

i = j = 0

result = []

while i < len(a) and j < len(b):
    if a[i] < b[j]:
        result.append(a[i])
        i += 1
    else:
        result.append(b[j])
        j += 1

while i < len(a):
    result.append(a[i])
    i += 1

while j < len(b):
    result.append(b[j])
    j += 1

print(result)
```

Time Complexity:

```text
O(n+m)
```

# 7. Majority Element

Question

Element appears more than n/2 times.

Input

```text
[2,2,1,1,2,2,2]
```

Output

```text
2
```

---

## Boyer Moore Voting Algorithm

Python

```python
arr = [2,2,1,1,2,2,2]

candidate = None
count = 0

for num in arr:
    if count == 0:
        candidate = num

    if num == candidate:
        count += 1
    else:
        count -= 1

print(candidate)
```

Time Complexity:

```text
O(n)
```

Space Complexity:

```text
O(1)
```

# 8. Trapping Rain Water

Input

```text
[0,1,0,2,1,0,1,3,2,1,2,1]
```

Output

```text
6
```

Concept:

```text
Water = min(left_max,right_max) - height
```

Optimized Two Pointer Solution

```python
def trap(height):
    left = 0
    right = len(height)-1

    left_max = 0
    right_max = 0

    water = 0

    while left < right:

        if height[left] < height[right]:

            if height[left] >= left_max:
                left_max = height[left]
            else:
                water += left_max - height[left]

            left += 1

        else:

            if height[right] >= right_max:
                right_max = height[right]
            else:
                water += right_max - height[right]

            right -= 1

    return water
```

Time Complexity:

```text
O(n)
```

Space Complexity:

```text
O(1)
```

# 9. Maximum Product Subarray

Input

```text
[2,3,-2,4]
```

Output

```text
6
```

Python

```python
nums = [2,3,-2,4]

max_prod = nums[0]
min_prod = nums[0]
result = nums[0]

for n in nums[1:]:

    if n < 0:
        max_prod, min_prod = min_prod, max_prod

    max_prod = max(n, max_prod*n)
    min_prod = min(n, min_prod*n)

    result = max(result, max_prod)

print(result)
```

Time Complexity:

```text
O(n)
```

# 10. Merge Intervals

Input

```text
[[1,3],[2,6],[8,10],[15,18]]
```

Output

```text
[[1,6],[8,10],[15,18]]
```

Python

```python
intervals = [[1,3],[2,6],[8,10],[15,18]]

intervals.sort()

merged = [intervals[0]]

for start,end in intervals[1:]:

    last_end = merged[-1][1]

    if start <= last_end:
        merged[-1][1] = max(last_end,end)
    else:
        merged.append([start,end])

print(merged)
```

Time Complexity:

```text
O(n log n)
```

# 11. Longest Consecutive Sequence

Input

```text
[100,4,200,1,3,2]
```

Output

```text
4
```

Sequence

```text
1,2,3,4
```

Python

```python
nums = [100,4,200,1,3,2]

s = set(nums)

longest = 0

for num in s:

    if num - 1 not in s:

        current = num
        streak = 1

        while current + 1 in s:
            current += 1
            streak += 1

        longest = max(longest, streak)

print(longest)
```

Time Complexity

```text
O(n)
```

# 12. Sliding Window Maximum

Input

```text
nums = [1,3,-1,-3,5,3,6,7]
k = 3
```

Output

```text
[3,3,5,5,6,7]
```

Python

```python
from collections import deque

nums = [1,3,-1,-3,5,3,6,7]
k = 3

dq = deque()
result = []

for i in range(len(nums)):

    while dq and dq[0] <= i-k:
        dq.popleft()

    while dq and nums[dq[-1]] < nums[i]:
        dq.pop()

    dq.append(i)

    if i >= k-1:
        result.append(nums[dq[0]])

print(result)
```

Time Complexity

```text
O(n)
```

Space Complexity

```text
O(k)
```
