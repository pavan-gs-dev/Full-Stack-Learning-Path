# Python Input, Output, Operators, and Type Conversion

---

# Python Input and Output

## What is Input?

Input is data provided by the user during program execution.

Python uses the `input()` function to accept user input.

### Example

```python
name = input("Enter your name: ")

print(name)
```

### Output

```text
Enter your name: Pavan
Pavan
```

---

## Input Always Returns a String

```python
age = input("Enter Age: ")

print(type(age))
```

### Output

```text
<class 'str'>
```

Even if the user enters a number, Python stores it as a string.

---

## Taking Integer Input

```python
age = int(input("Enter Age: "))

print(age)
print(type(age))
```

### Output

```text
Enter Age: 22
22
<class 'int'>
```

---

## Taking Float Input

```python
salary = float(input("Enter Salary: "))

print(salary)
```

### Output

```text
Enter Salary: 25000.50
25000.5
```

---

# What is Output?

Output is information displayed to the user.

Python uses the `print()` function.

### Example

```python
print("Hello World")
```

### Output

```text
Hello World
```

---

## Printing Multiple Values

```python
name = "Pavan"
age = 22

print(name, age)
```

### Output

```text
Pavan 22
```

---

## Formatted Output

### f-String

```python
name = "Pavan"
age = 22

print(f"My name is {name} and I am {age} years old.")
```

### Output

```text
My name is Pavan and I am 22 years old.
```

# Python Operators

Operators are symbols used to perform operations on values and variables.

---

# 1. Arithmetic Operators

| Operator | Meaning        |
| -------- | -------------- |
| +        | Addition       |
| -        | Subtraction    |
| *        | Multiplication |
| /        | Division       |
| //       | Floor Division |
| %        | Modulus        |
| **       | Exponent       |

### Example

```python
a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a // b)
print(a % b)
print(a ** b)
```

### Output

```text
13
7
30
3.3333333333333335
3
1
1000
```

---

# 2. Comparison Operators

Used to compare values.

| Operator | Meaning               |
| -------- | --------------------- |
| ==       | Equal                 |
| !=       | Not Equal             |
| >        | Greater Than          |
| <        | Less Than             |
| >=       | Greater Than or Equal |
| <=       | Less Than or Equal    |

### Example

```python
a = 10
b = 20

print(a == b)
print(a != b)
print(a > b)
print(a < b)
```

### Output

```text
False
True
False
True
```

---

# 3. Logical Operators

| Operator | Meaning     |
| -------- | ----------- |
| and      | Logical AND |
| or       | Logical OR  |
| not      | Logical NOT |

### Example

```python
a = True
b = False

print(a and b)
print(a or b)
print(not a)
```

### Output

```text
False
True
False
```

---

# 4. Assignment Operators

| Operator | Example |
| -------- | ------- |
| =        | x = 10  |
| +=       | x += 5  |
| -=       | x -= 5  |
| *=       | x *= 5  |
| /=       | x /= 5  |

### Example

```python
x = 10

x += 5

print(x)
```

### Output

```text
15
```

---

# 5. Membership Operators

Used to check if a value exists in a collection.

| Operator |
| -------- |
| in       |
| not in   |

### Example

```python
fruits = ["Apple", "Banana"]

print("Apple" in fruits)
print("Mango" not in fruits)
```

### Output

```text
True
True
```

---

# 6. Identity Operators

Used to compare memory references.

| Operator |
| -------- |
| is       |
| is not   |

### Example

```python
a = [1, 2]
b = a

print(a is b)
```

### Output

```text
True
```

---

# 7. Bitwise Operators

Works on binary numbers.

| Operator | Meaning     |
| -------- | ----------- |
| &        | AND         |
| |        | OR          |
| ^        | XOR         |
| ~        | NOT         |
| <<       | Left Shift  |
| >>       | Right Shift |

### Example

```python
a = 5
b = 3

print(a & b)
print(a | b)
```

# Python Type Conversion

Type Conversion means converting one data type into another.

---

# Implicit Type Conversion

Python automatically converts types.

### Example

```python
a = 10
b = 5.5

result = a + b

print(result)
print(type(result))
```

### Output

```text
15.5
<class 'float'>
```

Python converts `int` to `float` automatically.

---

# Explicit Type Conversion

Programmer manually converts data types.

---

## Convert String to Integer

```python
num = "100"

value = int(num)

print(value)
print(type(value))
```

### Output

```text
100
<class 'int'>
```

---

## Convert Integer to String

```python
age = 22

text = str(age)

print(text)
print(type(text))
```

### Output

```text
22
<class 'str'>
```

---

## Convert Integer to Float

```python
num = 10

value = float(num)

print(value)
```

### Output

```text
10.0
```

---

## Convert Float to Integer

```python
price = 99.99

value = int(price)

print(value)
```

### Output

```text
99
```

Note: Decimal part is removed.

---

## Convert String to Float

```python
salary = "25000.50"

value = float(salary)

print(value)
```

### Output

```text
25000.5
```

# Common Interview Questions

## What is the difference between input() and print()?

* `input()` receives data from the user.
* `print()` displays data to the user.

---

## What is Type Conversion?

Type Conversion is the process of converting one data type into another.

---

## Difference Between Implicit and Explicit Conversion?

### Implicit

Performed automatically by Python.

### Explicit

Performed manually using functions like:

```python
int()
float()
str()
bool()
```

---

# Interview Summary

```text
input()  -> Takes input from user

print()  -> Displays output

Operators -> Perform operations

Type Conversion -> Changes one data type to another
```

# One-Line Interview Answer

Python provides input() for user input, print() for output, operators for performing operations, and type conversion functions to convert data between different data types.
