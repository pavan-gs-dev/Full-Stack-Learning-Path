# 11.md - FastAPI APIs, Endpoints, Requests, Responses, Pydantic Validation, and API Documentation

# What is an API?

API stands for:

```text
Application Programming Interface
```

An API allows two applications to communicate.

Example:

```text
Mobile App
    │
    ▼
FastAPI API
    │
    ▼
Database
```

---

# Real Life Example

Imagine Zomato:

```text
Mobile App
```

User requests:

```text
Show Restaurants
```

The app sends a request to an API.

API returns:

```json
[
    {
        "id": 1,
        "name": "Restaurant A"
    }
]
```

---

# What is an Endpoint?

An Endpoint is a URL that performs a specific task.

Example:

```text
/users
/products
/orders
```

FastAPI Example:

```python
@app.get("/users")
def get_users():
    return {"message": "Users"}
```

Endpoint:

```text
/users
```

---


# What is a Request?

A Request is data sent by a client.

Example:

```text
Browser
Mobile App
Frontend
Postman
```

Request:

```http
GET /users
```

---

# What is a Response?

A Response is data returned by the server.

Example:

```json
{
    "name": "Pavan",
    "age": 22
}
```

---

# GET Endpoint

Used to fetch data.

```python
from fastapi import FastAPI

app = FastAPI()


@app.get("/users")
def users():

    return {
        "message": "Users List"
    }
```

---

# POST Endpoint

Used to create data.

```python
@app.post("/users")
def create_user():

    return {
        "message": "User Created"
    }
```

---

# PUT Endpoint

Used to update data.

```python
@app.put("/users")
def update_user():

    return {
        "message": "Updated"
    }
```

---

# DELETE Endpoint

Used to delete data.

```python
@app.delete("/users")
def delete_user():

    return {
        "message": "Deleted"
    }
```

---

# Path Parameters

Value comes from URL.

```python
@app.get("/users/{id}")
def get_user(id: int):

    return {
        "user_id": id
    }
```

URL:

```text
/users/10
```

Output:

```json
{
    "user_id": 10
}
```

---

# Query Parameters

```python
@app.get("/search")
def search(name: str):

    return {
        "name": name
    }
```

URL:

```text
/search?name=Pavan
```

Output:

```json
{
    "name": "Pavan"
}
```

# What is Pydantic?

Pydantic is a validation library used by FastAPI.

It checks:

```text
Data Type
Required Fields
Valid Data
```

Automatically.

---

# Why Pydantic?

Without Validation:

```python
{
    "name": 123
}
```

Wrong data may enter database.

Pydantic prevents this.

---

# First Pydantic Model

```python
from pydantic import BaseModel


class User(BaseModel):

    name: str
    age: int
```

---

# Using Pydantic

```python
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()


class User(BaseModel):

    name: str
    age: int


@app.post("/users")
def create_user(user: User):

    return user
```

# Required Fields

```python
class User(BaseModel):

    name: str
    age: int
```

Both fields required.

Request:

```json
{
    "name": "Pavan"
}
```

Error:

```text
age field required
```

---

# Optional Fields

```python
from typing import Optional


class User(BaseModel):

    name: str
    age: Optional[int] = None
```

Now age is optional.

---

# Validation Rules

```python
from pydantic import BaseModel, Field


class User(BaseModel):

    name: str = Field(
        min_length=3,
        max_length=50
    )

    age: int = Field(
        gt=0,
        lt=120
    )
```

Meaning:

```text
Name:
3-50 Characters

Age:
1-119
```

# What are {}

In FastAPI responses:

```python
return {
    "name": "Pavan"
}
```

This is a Python Dictionary.

JSON Response:

```json
{
    "name": "Pavan"
}
```

Used when returning a single object.

---

# What are []

Example:

```python
return [
    {
        "id": 1
    },
    {
        "id": 2
    }
]
```

This is a Python List.

JSON Response:

```json
[
    {
        "id": 1
    },
    {
        "id": 2
    }
]
```

Used for multiple records.

Example:

```text
Users
Products
Orders
```

---

# What are () ?

Example:

```python
data = (
    "Pavan",
    22
)
```

This is a Tuple.

Normally:

```text
Not returned in APIs
```

FastAPI converts it automatically if needed.

Most APIs use:

```text
Dictionary {}
List []
```

# Full Example

```python
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()


class User(BaseModel):

    name: str
    age: int


@app.post("/users")
def create_user(user: User):

    return {
        "message": "User Created",
        "user": user
    }
```

Request:

```json
{
    "name": "Pavan",
    "age": 22
}
```

Response:

```json
{
    "message": "User Created",
    "user": {
        "name": "Pavan",
        "age": 22
    }
}
```
