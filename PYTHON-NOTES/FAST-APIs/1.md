# 10.md - FastAPI Introduction, Installation, Setup, and Project Structure

# What is FastAPI?

FastAPI is a modern Python web framework used to build:

* REST APIs
* Backend Applications
* AI APIs
* Machine Learning APIs
* Authentication Systems
* Microservices
* Chat Applications

FastAPI was created by:

```text
Sebastián Ramírez
```

---

# Why FastAPI?

Before FastAPI, developers used:

* Flask
* Django

FastAPI became popular because it is:

✅ Fast

✅ Easy to Learn

✅ Automatic API Documentation

✅ Built-in Validation

✅ Async Support

✅ Modern Python Support

---

```text
ChatGPT APIs

Mobile App Backends
```

# FastAPI vs Flask

| Feature       | FastAPI   | Flask          |
| ------------- | --------- | -------------- |
| Speed         | Very Fast | Fast           |
| Validation    | Built-in  | Manual         |
| API Docs      | Automatic | Extra Packages |
| Type Hints    | Excellent | Limited        |
| Async Support | Yes       | Limited        |

# First FastAPI Application

Create:

```text
main.py
```

```python
from fastapi import FastAPI

app = FastAPI()


@app.get("/")
def home():

    return {
        "message": "Hello FastAPI"
    }
```

---

# Run Application

```bash
uvicorn main:app --reload
```

Meaning:

```text
main
 ↓
main.py

app
 ↓
FastAPI Object
```

# Creating Multiple Routes

```python
from fastapi import FastAPI

app = FastAPI()


@app.get("/")
def home():

    return {
        "message": "Home Page"
    }


@app.get("/about")
def about():

    return {
        "message": "About Page"
    }
```

---

# Path Parameters

```python
from fastapi import FastAPI

app = FastAPI()


@app.get("/user/{id}")
def get_user(id: int):

    return {
        "user_id": id
    }
```

URL:

```text
/user/10
```

Output:

```json
{
    "user_id": 10
}
```

---

# Query Parameters

URL:


# Common Interview Questions

## What is FastAPI?

A modern Python framework used to build APIs.

---

## Why FastAPI?

* High Performance
* Automatic Validation
* Automatic Documentation

---

## What is Uvicorn?

An ASGI server used to run FastAPI applications.

---

## How do you run FastAPI?

```bash
uvicorn main:app --reload
```

---

## What is Swagger UI?

Automatic API documentation.

URL:

```text
/docs
```

---

## Difference Between GET and POST?

| GET            | POST         |
| -------------- | ------------ |
| Read Data      | Create Data  |
| URL Parameters | Request Body |

---

## What is a Route?

A URL endpoint handled by FastAPI.

Example:

```python
@app.get("/users")
```
