# Introduction

File Handling is used to store data permanently in files instead of keeping it only in memory.

Without files:

```python id="gvpcll"
name = "Pavan"
```

Data exists only while the program is running.

When the program closes:

```text id="8h8lbk"
Data Lost
```

Using files:

```text id="0wtdgz"
Data Saved Permanently
```

---

File handling is used for:

# Opening a File

Python uses the `open()` function.

Syntax:

```python id="5xbmqy"
file = open("filename", "mode")
```

Example:

```python id="1xibys"
file = open("data.txt", "r")
```

---

# File Modes

| Mode | Meaning         |
| ---- | --------------- |
| r    | Read            |
| w    | Write           |
| a    | Append          |
| x    | Create New File |
| rb   | Read Binary     |
| wb   | Write Binary    |
| r+   | Read and Write  |

---

# Read Mode (r)

Used to read existing files.

Example:

```python id="b24n9w"
file = open("data.txt", "r")

content = file.read()

print(content)

file.close()
```

---

# write() Method

Used to write data to a file.

Example:

```python id="3m7vfh"
file = open("data.txt", "w")

file.write("Hello Python")

file.close()
```

Output File:

```text id="tx9avc"
Hello Python
```

---

# Important Note About Write Mode

```python id="3tq5hm"
open("data.txt", "w")
```

Write mode:

* Creates file if it doesn't exist
* Removes existing content if file exists

Example:

Before:

```text id="g54fxf"
Hello
```

After:

```python id="woc2cc"
file = open("data.txt", "w")
file.write("Python")
```

File:

```text id="bdm7a6"
Python
```

---

# Append Mode (a)

Adds content without deleting existing data.

Example:

```python id="3gcywg"
file = open("data.txt", "a")

file.write("\nNew Line")

file.close()
```

Before:

```text id="9dsyw0"
Hello
```

After:

```text id="67u2j9"
Hello
New Line
```

# Closing Files

Always close files after use.

```python id="ch4bd6"
file.close()
```

Reason:

* Releases memory
* Prevents corruption
* Avoids resource leaks

---

# Using with Statement

Recommended approach.

Example:

```python id="k9yqzd"
with open("data.txt", "r") as file:

    content = file.read()

    print(content)
```

No need for:

```python id="v5x58r"
file.close()
```

Python automatically closes the file.

# Python Dictionary to JSON

Python:

```python id="x9jafw"
import json

student = {
    "name": "Pavan",
    "age": 22
}

json_data = json.dumps(student)

print(json_data)
```

Output:

```json
{"name": "Pavan", "age": 22}
```

---

# JSON to Dictionary

```python id="3xd3sv"
import json

json_data = '{"name":"Pavan","age":22}'

data = json.loads(json_data)

print(data)
```

Output:

```python id="7k89wo"
{'name': 'Pavan', 'age': 22}
```

---

# Write JSON to File

```python id="o0vhsd"
import json

student = {
    "name": "Pavan",
    "age": 22
}

with open("student.json", "w") as file:

    json.dump(student, file)
```

Creates:

```json
{
  "name": "Pavan",
  "age": 22
}
```

---

# Read JSON File

```python id="16uxph"
import json

with open("student.json", "r") as file:

    data = json.load(file)

print(data)
```

Output:

```python id="l0c4wq"
{'name': 'Pavan', 'age': 22}
```

---

# Pretty Print JSON

```python id="znr14l"
import json

student = {
    "name": "Pavan",
    "age": 22
}

print(
    json.dumps(
        student,
        indent=4
    )
)
```

Output:

```json
{
    "name": "Pavan",
    "age": 22
}
```

# File Handling Interview Questions

## What is File Handling?

File handling is the process of creating, reading, writing, updating, and deleting files.

---

## What does open() do?

Used to open a file.

---

## Difference Between Read and Write Mode?

| Read            | Write        |
| --------------- | ------------ |
| Reads file      | Writes file  |
| File must exist | Creates file |
| Cannot modify   | Can modify   |

---

## Difference Between Write and Append?

| Write            | Append         |
| ---------------- | -------------- |
| Overwrites file  | Adds content   |
| Deletes old data | Keeps old data |

---

## Why Use with open()?

Automatically closes files after use.

---

# JSON Interview Questions

## What is JSON?

JSON is a lightweight data exchange format.

---

## What is json.dump()?

Writes Python data to a JSON file.

---

## What is json.load()?

Reads JSON data from a file.

---

## Difference Between dump() and dumps()?

| Function | Purpose                       |
| -------- | ----------------------------- |
| dump()   | Write JSON to File            |
| dumps()  | Convert Object to JSON String |

---

## Difference Between load() and loads()?

| Function | Purpose                              |
| -------- | ------------------------------------ |
| load()   | Read JSON File                       |
| loads()  | Convert JSON String to Python Object |
