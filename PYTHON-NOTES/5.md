# Python Functions

## What is a Function?

A Function is a reusable block of code that performs a specific task.

Instead of writing the same code multiple times, we create a function and call it whenever needed.

### Why Functions?

Functions help us:

* Reduce code duplication
* Improve readability
* Improve maintainability
* Reuse code
* Break large programs into smaller modules

---

# Real Life Example

Imagine a calculator.

Instead of writing addition logic everywhere:

```python id="6v5p5q"
print(10 + 20)
print(30 + 40)
print(50 + 60)
```

We can create a function:

```python id="ch4hm7"
def add(a, b):
    return a + b
```

Now we can reuse it:

```python id="h1ojql"
print(add(10, 20))
print(add(30, 40))
print(add(50, 60))
```

# Function Execution Flow

```python id="zn4xt0"
def greet():
    print("Hello")

print("Start")

greet()

print("End")
```

Output:

```text id="gf4l5e"

Hello
End
```

Execution:

1. Start
2. Function call
3. Function body executes
4. Returns control
5. End

# Return Value

A function can return a value using the `return` keyword.

---

# Example

```python id="5qjs1l"
def add(a, b):
    return a + b

result = add(10, 20)

print(result)
```

Output:

```text id="2a6xuo"
30
```

---

# Why Return is Important?

Without return:

```python id="w7r6ue"
def add(a, b):
    print(a + b)
```

Cannot reuse the result.

With return:

```python id="c5pw8k"
result = add(10, 20)
```

The value can be stored, reused, or passed to another function.

---

# Returning Multiple Values

```python id="93l4rf"
def calculate(a, b):
    return a+b, a-b

result = calculate(10, 5)

print(result)
```

Output:

```text id="jx10qs"
(15, 5)
```

# Local Variables

Variables created inside a function.

```python id="vdp30r"
def show():
    x = 10
```

Accessible only inside the function.

---

# Global Variables

Variables created outside a function.

```python id="78on4g"
x = 10

def show():
    print(x)
```

Accessible throughout the program.

---

# Lambda Functions

A Lambda Function is a small anonymous function.

Anonymous means it has no name.

---

# Normal Function

```python id="uw5yks"
def square(x):
    return x * x
```

---

# Lambda Version

```python id="f0pq3u"
square = lambda x: x * x
```

---

# Example

```python id="9wnlci"
square = lambda x: x * x

print(square(5))
```

Output:

```text id="32h6h6"
25
```

---

# Lambda With Multiple Arguments

```python id="y6bjdt"
add = lambda a, b: a + b

print(add(10, 20))
```

Output:

```text id="f7krhl"
30
```

---

# Why Use Lambda?

Used when:

* Small functions are needed
* One-time use functions
* Sorting
* Filtering
* Mapping

---

# Lambda With sorted()

```python id="s7w7jr"
students = [
    ("Pavan", 22),
    ("Rahul", 20),
    ("Arjun", 25)
]

students.sort(key=lambda x: x[1])

print(students)
```

Output:

```text id="8hrnng"
[('Rahul', 20), ('Pavan', 22), ('Arjun', 25)]
```

# Common Interview Questions

## What is a Function?

## Difference Between Parameter and Argument?

Parameters are variables in function definition.

Arguments are actual values passed during function call.

---

## What is Return?

Return sends a value back to the caller.

---

## What is Lambda Function?

A small anonymous function defined using the `lambda` keyword.

---

## Difference Between Normal Function and Lambda?

| Normal Function     | Lambda            |
| ------------------- | ----------------- |
| Uses def            | Uses lambda       |
| Multiple statements | Single expression |
| Has name            | Anonymous         |

---

## What is *args?

Accepts multiple positional arguments.

---

## What is **kwargs?

Accepts multiple keyword arguments.
