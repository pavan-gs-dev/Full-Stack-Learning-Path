# Python File Handling and JSON

## Introduction

File Handling is used to store data permanently in files.

Normally, when a Python program ends, all variables are removed from memory.

Example:

```python
name = "Pavan"
```

When the program stops, the value is lost.

To save data permanently, we use files.

Examples:

* User data
* Configuration files
* Reports
* Logs
* Student records
* Product data

---

# What is a File?

A file is a container used to store data on a storage device.

Examples:

```text
student.txt
users.json
config.json
data.csv
report.pdf
```

---

# Why File Handling?

File Handling helps us:

* Save data permanently
* Read existing data
* Update data
* Delete data
* Share data between programs

---

# Opening a File

Python uses the `open()` function.

Syntax:

```python
open(filename, mode)
```

Example:

```python
file = open("data.txt", "r")
```

---

# File Modes

| Mode | Description   |
| ---- | ------------- |
| r    | Read          |
| w    | Write         |
| a    | Append        |
| x    | Create        |
| r+   | Read + Write  |
| w+   | Write + Read  |
| a+   | Append + Read |
| rb   | Read Binary   |
| wb   | Write Binary  |

---

# Read Mode (r)

Used to read an existing file.

Example:

```python
file = open("data.txt", "r")

content = file.read()

print(content)

file.close()
```

---

# What Does read() Do?

Reads the entire file.

Example:

```text
Hello
Python
```

Output:

```text
Hello
Python
```

---

# Read Specific Characters

```python
file = open("data.txt", "r")

print(file.read(5))

file.close()
```

Output:

```text
Hello
```

---

# readline()

Reads one line at a time.

Example:

```python
file = open("data.txt", "r")

print(file.readline())

file.close()
```

---

# readlines()

Returns all lines as a list.

```python
file = open("data.txt", "r")

print(file.readlines())

file.close()
```

Output:

```python
['Hello\n', 'Python\n']
```

---

# Write Mode (w)

Creates a new file.

If the file already exists, old data is deleted.

Example:

```python
file = open("data.txt", "w")

file.write("Hello Python")

file.close()
```

---

# Important

```python
open("data.txt", "w")
```

Deletes old content before writing.

---

# Append Mode (a)

Adds new content without deleting existing content.

Example:

```python
file = open("data.txt", "a")

file.write("\nNew Line")

file.close()
```

---

# Create File (x)

Creates a file.

Raises error if file already exists.

```python
file = open("newfile.txt", "x")

file.close()
```

---

# Closing Files

Always close files after use.

```python
file.close()
```

Reason:

* Frees memory
* Prevents corruption
* Improves performance

---

# Using with Statement

Recommended approach.

Python automatically closes the file.

```python
with open("data.txt", "r") as file:

    print(file.read())
```

No need:

```python
file.close()
```

---

# Writing Multiple Lines

```python
with open("data.txt", "w") as file:

    file.write("Line 1\n")
    file.write("Line 2\n")
    file.write("Line 3\n")
```

---

# File Exists Check

```python
import os

if os.path.exists("data.txt"):
    print("File Found")
```

---

# Delete File

```python
import os

os.remove("data.txt")
```

---

# Practical Example

Save Student Data

```python
with open("student.txt", "w") as file:

    file.write("Name: Pavan\n")
    file.write("Age: 22\n")
```

# JSON Files

## What is JSON?

JSON stands for:

```text
JavaScript Object Notation
```

JSON is the most popular format used to exchange data.

Used in:

* APIs
* Web Applications
* Mobile Applications
* Databases
* Configuration Files

---

# JSON Example

```json
{
    "name": "Pavan",
    "age": 22,
    "city": "Bangalore"
}
```

---

# Why JSON?

JSON is:

* Lightweight
* Easy to Read
* Easy to Write
* Language Independent

---

# Python JSON Module

Import:

```python
import json
```

---

# Convert Dictionary to JSON

Python Object:

```python
student = {
    "name": "Pavan",
    "age": 22
}
```

Convert:

```python
import json

json_data = json.dumps(student)

print(json_data)
```

Output:

```json
{"name": "Pavan", "age": 22}
```

---

# dumps()

Converts Python Object → JSON String

```python
json.dumps()
```

---

# Convert JSON to Dictionary

```python
import json

data = '{"name":"Pavan","age":22}'

student = json.loads(data)

print(student)
```

Output:

```python
{'name': 'Pavan', 'age': 22}
```

---

# loads()

Converts JSON String → Python Object

```python
json.loads()
```

---

# Save JSON to File

```python
import json

student = {
    "name": "Pavan",
    "age": 22
}

with open("student.json", "w") as file:

    json.dump(student, file)
```

Creates:

```json
{
    "name": "Pavan",
    "age": 22
}
```

---

# dump()

Python Object → JSON File

```python
json.dump()
```

---

# Read JSON File

```python
import json

with open("student.json", "r") as file:

    data = json.load(file)

print(data)
```

Output:

```python
{'name': 'Pavan', 'age': 22}
```

---

# load()

JSON File → Python Object

```python
json.load()
```

---

# Pretty JSON Formatting

```python
import json

student = {
    "name": "Pavan",
    "age": 22
}

print(
    json.dumps(
        student,
        indent=4
    )
)
```

Output:

```json
{
    "name": "Pavan",
    "age": 22
}
```

# Common Interview Questions

## What is File Handling?

File Handling is the process of reading, writing, updating, and deleting files.

---

## Difference Between read() and readline()?

| read()      | readline() |
| ----------- | ---------- |
| Entire file | One line   |

---

## Difference Between w and a Mode?

| w                   | a                 |
| ------------------- | ----------------- |
| Overwrites file     | Appends data      |
| Deletes old content | Keeps old content |

---

## What is JSON?

JSON is a lightweight data-interchange format used to store and exchange structured data.

---

## Difference Between dump() and dumps()?

| dump()              | dumps()                 |
| ------------------- | ----------------------- |
| Writes JSON to file | Converts to JSON string |

---

## Difference Between load() and loads()?

| load()          | loads()           |
| --------------- | ----------------- |
| Reads JSON file | Reads JSON string |

---

# Interview Summary

```text
open()      -> Open File

read()      -> Read Entire File

readline()  -> Read One Line

write()     -> Write Data

append()    -> Add Data

close()     -> Close File

json.dump() -> Save JSON File

json.load() -> Read JSON File

json.dumps() -> Object to JSON String

json.loads() -> JSON String to Object
```

# One-Line Interview Answer

File Handling allows Python programs to store and retrieve data from files, while JSON provides a structured and widely used format for storing and exchanging data between applications.
