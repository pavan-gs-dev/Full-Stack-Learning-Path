# 7a.md - Python @staticmethod, @classmethod, ABC, and @abstractmethod

# Introduction

When learning Python OOP, many beginners get confused by:

```python
@staticmethod

@classmethod

from abc import ABC, abstractmethod
```

These concepts are important for interviews and real-world projects.

This chapter explains them in simple language.

---

# What is @staticmethod?

A Static Method is a method that belongs to a class but does not use:

```python
self
```

or

```python
cls
```

---

# Why Use Static Methods?

Sometimes we need a function inside a class that does not depend on:

* Object data
* Class data

Example:

Adding two numbers.

---

# Example Without Static Method

```python
class Calculator:

    def add(self, a, b):

        return a + b
```

Create Object:

```python
obj = Calculator()

print(obj.add(10, 20))
```

Output:

```text
30
```

---

# Example Using Static Method

```python
class Calculator:

    @staticmethod
    def add(a, b):

        return a + b
```

Call Directly:

```python
print(Calculator.add(10, 20))
```

Output:

```text
30
```

---

# Why Static Method Here?

Because:

```text
Addition does not need object data.
```

No need for:

```python
self
```

No need for:

```python
cls
```

---

# Real-Life Example

Imagine:

```text
Calculator
```

Functions:

```text
Add
Subtract
Multiply
Divide
```

These operations do not need object information.

Static methods are perfect here.

---

# Interview Answer

A static method belongs to the class and does not require self or cls.

---

# What is @classmethod?

A Class Method works with class variables.

Instead of:

```python
self
```

it uses:

```python
cls
```

---

# What is cls?

```python
cls
```

means:

```text
Current Class
```

---

# Example

```python
class School:

    school_name = "ABC School"

    @classmethod
    def show_school(cls):

        print(cls.school_name)
```

Call:

```python
School.show_school()
```

Output:

```text
ABC School
```

---

# Why Use Class Methods?

Because:

```python
school_name
```

belongs to the class.

Not to an object.

---

# Real-Life Example

```text
School Name
Principal Name
School Address
```

These values are shared by all students.

Therefore:

```python
@classmethod
```

is useful.

---

# Difference Between self and cls

## self

Represents:

```text
Current Object
```

Example:

```python
self.name
```

Each object has its own value.

---

## cls

Represents:

```text
Current Class
```

Example:

```python
cls.school_name
```

Shared by all objects.

---

# Example

```python
class Student:

    school = "ABC School"

    def __init__(self, name):

        self.name = name
```

Memory:

```text
Student Class
    school

Object 1
    name = Pavan

Object 2
    name = Rahul
```

---

# Interview Answer

A class method uses cls and can access or modify class variables.

---

# Easy Memory Trick

```text
self → Current Object

cls → Current Class
```

---

# What is ABC?

ABC stands for:

```text
Abstract Base Class
```

Import:

```python
from abc import ABC
```

---

# Why Use ABC?

Suppose we create:

```text
Vehicle
```

Every vehicle should have:

```text
start()
```

But every vehicle starts differently.

Example:

```text
Car

Bike

Bus

Truck
```

Each must implement:

```python
start()
```

ABC helps enforce this rule.

---

# Abstract Class Example

```python
from abc import ABC

class Vehicle(ABC):

    pass
```

Now:

```text
Vehicle
```

becomes an Abstract Class.

---

# What is @abstractmethod?

An Abstract Method is a method that must be implemented by child classes.

Import:

```python
from abc import abstractmethod
```

---

# Example

```python
from abc import ABC, abstractmethod


class Vehicle(ABC):

    @abstractmethod
    def start(self):

        pass
```

Meaning:

```text
Every child class MUST create start().
```

---

# Child Class Example

```python
class Car(Vehicle):

    def start(self):

        print("Car Started")
```

Create Object:

```python
car = Car()

car.start()
```

Output:

```text
Car Started
```

---

# What Happens if Child Class Does Not Implement Method?

```python
class Car(Vehicle):

    pass
```

Python Error:

```text
Can't instantiate abstract class
```

Reason:

```text
start() was not implemented.
```

---

# Real-Life Example

Imagine a school form.

Every student must fill:

```text
Name

Age

Phone Number
```

Mandatory.

Similarly:

```python
@abstractmethod
```

creates mandatory methods.

---

# Complete Example

```python
from abc import ABC, abstractmethod


class Vehicle(ABC):

    @abstractmethod
    def start(self):

        pass


class Car(Vehicle):

    def start(self):

        print("Car Started")


class Bike(Vehicle):

    def start(self):

        print("Bike Started")


car = Car()

bike = Bike()

car.start()

bike.start()
```

Output:

```text
Car Started

Bike Started
```

---

# When to Use Abstract Classes?

Use Abstract Classes when:

* Parent class defines rules
* Child classes must follow those rules
* Large projects
* Framework development
* Enterprise applications

---

# Difference Between Static Method and Class Method

| Static Method    | Class Method             |
| ---------------- | ------------------------ |
| Uses no self     | Uses cls                 |
| Uses no cls      | Accesses class variables |
| Utility Function | Works with class data    |
| @staticmethod    | @classmethod             |

---

# Difference Between Class Method and Normal Method

| Normal Method          | Class Method              |
| ---------------------- | ------------------------- |
| Uses self              | Uses cls                  |
| Works with object data | Works with class data     |
| Requires object        | Can be called using class |

---

# Difference Between Abstract Class and Normal Class

| Normal Class       | Abstract Class                 |
| ------------------ | ------------------------------ |
| Can create objects | Cannot create objects directly |
| Optional methods   | Mandatory methods possible     |
| Uses class keyword | Uses ABC                       |
