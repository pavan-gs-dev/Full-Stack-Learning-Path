# Python Variables

## What is a Variable?

A variable is a name used to store a value in memory.

### Example

```python
name = "Pavan"
age = 22
```

Here:

* `name` stores `"Pavan"`
* `age` stores `22`

---

# Deep Understanding of Variables

Think of computer memory as a large warehouse.

When you write:

```python
age = 22
```

Python performs the following steps:

### Step 1: Create the Object

Python creates the integer object `22` in memory.

### Step 2: Create the Variable

Python creates the variable `age`.

### Step 3: Assign Reference

The variable `age` points to the memory location of the object `22`.

```
Variable          Object
--------          ------
age  --------->   22
```

**Important:**

Python variables do not store values directly.

They store references to objects.

---

# Variable Assignment Example

```python
a = 10
b = a
```

Memory Representation:

```
a ----\
       ---> 10
b ----/
```

Both variables point to the same object.

---

# Reassignment Example

```python
a = 10
b = a

a = 20
```

Now:

```
a -------> 20

b -------> 10
```

Python creates a new object `20` and updates the reference of `a`.

---

# Variables and Memory

Example:

```python
name = "Pavan"
```

Memory:

```
Memory Address     Value
-------------      -----
1001               "Pavan"
```

Variable Table:

```
Variable     Address
--------     -------
name         1001
```

Representation:

```
name ---> "Pavan"
```

---

# Dynamic Typing in Python

Python is dynamically typed.

```python
x = 10
x = "Hello"
```

Initially:

```
x ---> 10
```

Later:

```
x ---> "Hello"
```

No type declaration is required.

---

# Types of Variables

## Local Variable

```python
def show():
    x = 10
```

Accessible only inside the function.

---

## Global Variable

```python
x = 10

def show():
    print(x)
```

Accessible throughout the program.

---

## Instance Variable

```python
class Student:
    def __init__(self):
        self.name = "Pavan"
```

Belongs to an object.

---

## Class Variable

```python
class Student:
    college = "ABC College"
```

Shared by all objects of the class.

# Python Interview Questions

## 1. What is a Variable?

**Answer:**

A variable is a name that refers to a value stored in memory.

---

## 2. Does Python Store Values or References?

**Answer:**

Python variables store references to objects, not the actual values.

---

## 3. What is Dynamic Typing?

**Answer:**

A variable can hold different types of data during program execution.

```python
x = 10
x = "Python"
```

---

## 4. Difference Between Variable and Object

| Variable         | Object           |
| ---------------- | ---------------- |
| Name             | Actual Data      |
| Reference Holder | Stored in Memory |

Example:

```python
x = 100
```

* `x` → Variable
* `100` → Object

---

## 5. What is the `id()` Function?

```python
x = 10
print(id(x))
```

Returns the unique identity (memory reference) of an object.

---

## 6. What Happens Internally When We Write?

```python
a = 10
```

Steps:

1. Create object `10`
2. Create variable `a`
3. Assign object reference to variable

---

## 7. What is Garbage Collection?

```python
a = 10
a = 20
```

If no variable references the object `10`, Python automatically removes it using Garbage Collection.

---

## 8. Can Two Variables Point to the Same Object?

Yes.

```python
a = 10
b = a
```

Both variables point to the same object.

# Mutable Object Example

```python
a = [1, 2, 3]
b = a

b.append(4)

print(a)
```

Output:

```python
[1, 2, 3, 4]
```

Memory Representation:

```
a ----\
       ---> [1, 2, 3, 4]
b ----/
```

Both variables reference the same list object.

---

# Key Interview Points

* Variables are references to objects.
* Python uses dynamic typing.
* Variables do not store actual values directly.
* Objects are stored in memory.
* Python manages memory automatically.
* Garbage Collection removes unused objects.
* Multiple variables can reference the same object.

---

# One-Line Interview Answer

> A variable is a named reference to an object in memory. Python variables store references to objects rather than actual values, and memory management is handled automatically by Python.

# Python Data Types vs Data Structures

## Introduction

Many beginners get confused between **Data Types** and **Data Structures**.

A simple way to remember:

* **Data Type** → What kind of data is stored?
* **Data Structure** → How is data organized and stored?

---

# What is a Data Type?

A Data Type defines the type of value an object contains.

Examples:

```python
age = 22
price = 99.99
name = "Pavan"
is_student = True
```

| Variable   | Value | Data Type |
| ---------- | ----- | --------- |
| age        | 22    | int       |
| price      | 99.99 | float     |
| name       | Pavan | str       |
| is_student | True  | bool      |

---

# Python Built-in Data Types

## Numeric Types

### Integer (int)

Stores whole numbers.

```python
age = 22
marks = 100
```

### Float

Stores decimal numbers.

```python
price = 99.99
height = 5.8
```

### Complex

Stores complex numbers.

```python
x = 3 + 4j
```

---

## String Type

Stores text data.

```python
name = "Pavan"
city = "Bangalore"
```

---

## Boolean Type

Stores True or False.

```python
is_logged_in = True
is_admin = False
```

---

## Binary Types

### Bytes

```python
data = b"Hello"
```

### Bytearray

```python
data = bytearray(5)
```

### Memoryview

```python
data = memoryview(bytes(5))
```

---

## NoneType

Represents the absence of a value.

```python
x = None
```

---

# What is a Data Structure?

A Data Structure is a way of organizing and storing multiple values efficiently.

Examples:

```python
students = ["Pavan", "Rahul", "Manoj"]

marks = {
    "Math": 95,
    "Science": 90
}
```

---

# Python Built-in Data Structures

## List

Stores multiple values.

```python
fruits = ["Apple", "Banana", "Mango"]
```

### Features

* Ordered
* Mutable
* Allows Duplicates

### Example

```python
fruits.append("Orange")
```

Output:

```python
['Apple', 'Banana', 'Mango', 'Orange']
```

---

## Tuple

Stores multiple values.

```python
data = (10, 20, 30)
```

### Features

* Ordered
* Immutable
* Allows Duplicates

---

## Set

Stores unique values.

```python
numbers = {1, 2, 3}
```

### Features

* Unordered
* Mutable
* No Duplicates

Example:

```python
numbers = {1, 1, 2, 3}

print(numbers)
```

Output:

```python
{1, 2, 3}
```

---

## Frozenset

Immutable version of Set.

```python
data = frozenset([1, 2, 3])
```

### Features

* Unordered
* Immutable
* No Duplicates

---

## Dictionary

Stores data in Key-Value pairs.

```python
student = {
    "name": "Pavan",
    "age": 22
}
```

### Features

* Key-Value Pairs
* Mutable
* Fast Lookup

### Accessing Values

```python
print(student["name"])
```

Output:

```python
Pavan
```

---

# Difference Between Data Types and Data Structures

| Data Types                        | Data Structures                  |
| --------------------------------- | -------------------------------- |
| Define the type of a single value | Organize multiple values         |
| Tell what data is stored          | Tell how data is stored          |
| Examples: int, float, str, bool   | Examples: list, tuple, set, dict |
| Basic building blocks             | Built using data types           |
