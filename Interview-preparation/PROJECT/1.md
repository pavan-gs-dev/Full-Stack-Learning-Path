# 1. AI Face Finder

## Tell Me About Your Project

I developed an AI Face Finder web application using Python, Flask, OpenCV, and the face_recognition library.

The main goal of the project is to help photographers and event organizers easily share photos with users.

Normally after an event, photographers may have thousands of photos and users need to manually search for their pictures. My application automates this process using face recognition.

Photographers upload event photos to the platform. Users upload a selfie. The system extracts facial features from the selfie and compares them with all uploaded event photos. Matching photos are then displayed to the user.

This saves time and eliminates manual searching.

## Interview Questions

### Why did you choose Flask?

Flask is lightweight, simple, and easy to integrate with Python AI libraries.

---

### How does face recognition work?

The system extracts facial features from an image and converts them into numerical encodings. These encodings are compared with other faces to find matches.

---

### What is OpenCV?

OpenCV is an open-source computer vision library used for image processing.

---

### What happens when a user uploads a selfie?

The application generates a face encoding from the selfie and compares it with event photo encodings to find matching images.
