# Python Interview Questions and Answers for Freshers

---

# Python Basics

### Q1: What is Python?

**Answer:**
Python is a high-level, interpreted, object-oriented programming language known for its simple syntax and readability.

---

### Q2: What are the features of Python?

**Answer:**

* Easy to learn
* Interpreted language
* Object-oriented
* Cross-platform
* Large standard library
* Open source

---

### Q3: What is the difference between a compiler and an interpreter?

**Answer:**

* Compiler translates the entire program at once.
* Interpreter executes code line by line.
* Python uses an interpreter.

---

### Q4: What are variables?

**Answer:**
Variables are used to store data values in memory.

Example:

```python
name = "Pavan"
age = 22
```

### Q5: What are Python data types?

**Answer:**

* int
* float
* str
* bool
* list
* tuple
* set
* dictionary

---

# Control Flow

### Q6: What is a conditional statement?

**Answer:**
A conditional statement executes code based on a condition.

Example:

```python
if age >= 18:
    print("Adult")
```

### Q7: Difference between if, elif, and else?

**Answer:**

* if → first condition
* elif → additional condition
* else → default block

---

### Q8: Difference between for loop and while loop?

**Answer:**

**for loop**

* Used when number of iterations is known.

**while loop**

* Used when condition controls execution.

---

### Q9: What are break and continue?

**Answer:**

**break**

* Terminates the loop.

**continue**

* Skips current iteration.

---

### Q10: What is pass statement?

**Answer:**
pass is a placeholder statement that does nothing.

Example:

```python
if True:
    pass
```

---

# Functions

### Q11: What is a function?

**Answer:**
A function is a reusable block of code that performs a specific task.

---

### Q12: Difference between parameter and argument?

**Answer:**

Parameter:

```python
def add(a, b):
```

Argument:

```python
add(10, 20)
```

---

### Q13: Difference between return and print?

**Answer:**

print()

* Displays output.

return

* Sends value back to caller.

---

### Q14: What is recursion?

**Answer:**
A function calling itself is called recursion.

Example:

```python
def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n-1)
```

---

### Q15: What is lambda function?

**Answer:**
A lambda function is an anonymous one-line function.

Example:

```python
square = lambda x: x*x
```

---

# Python Data Structures

### Q16: What is a List?

**Answer:**
A list is an ordered and mutable collection.

Example:

```python
numbers = [1,2,3]
```

---

### Q17: What is a Tuple?

**Answer:**
A tuple is an ordered and immutable collection.

Example:

```python
data = (1,2,3)
```

---

### Q18: Difference between List and Tuple?

**Answer:**

List:

* Mutable

Tuple:

* Immutable

---

### Q19: What is a Set?

**Answer:**
A set is an unordered collection of unique elements.

Example:

```python
s = {1,2,3}
```

---

### Q20: What is a Dictionary?

**Answer:**
A dictionary stores data in key-value pairs.

Example:

```python
student = {
    "name":"Pavan",
    "age":22
}
```

---

### Q21: Difference between List and Set?

**Answer:**

List:

* Ordered
* Allows duplicates

Set:

* Unordered
* No duplicates

---

### Q22: What is list comprehension?

**Answer:**
A concise way to create lists.

Example:

```python
squares = [x*x for x in range(5)]
```

### Q24: What is a Class?

**Answer:**
A class is a blueprint for creating objects.

---

### Q25: What is an Object?

---

### Q29: What is Polymorphism?

---

### Q30: What is Abstraction?

---

### Q31: What is the difference between Class and Object?

# File Handling

### Q32: What is File Handling?

**Answer:**
File handling is used to create, read, write, and update files.

---

### Q33: How to open a file?

**Answer:**

```python
file = open("data.txt","r")
```

---

### Q34: File Modes?

**Answer:**

* r → Read
* w → Write
* a → Append
* x → Create

---

### Q35: Why use with open()?

**Answer:**
It automatically closes the file.

Example:

```python
with open("data.txt","r") as file:
    print(file.read())
```

---

### Q36: Difference between write() and append()?

**Answer:**

write()

* Replaces existing content.

append()

* Adds content to end of file.

---

# Exception Handling

### Q37: What is an Exception?

**Answer:**
An exception is a runtime error that interrupts program execution.

---

### Q38: What is Exception Handling?

**Answer:**
Exception handling prevents program crashes and handles errors gracefully.

---

### Q39: What are try and except?

**Answer:**

try:

* Contains risky code.

except:

* Handles the error.

Example:

```python
try:
    print(10/0)
except ZeroDivisionError:
    print("Cannot divide by zero")
```

---

### Q40: What is finally?

**Answer:**
finally block always executes whether exception occurs or not.

---

### Q41: What is raise?

**Answer:**
raise is used to manually generate exceptions.

Example:

```python
raise ValueError("Invalid value")
```

---

Example:

```python

```

---

### Q43: What is a Package?

---

### Q44: Difference between Module and Package?

**Answer:**

* Single Python file

Package:

* Collection of modules

---

### Q45: What is import?

**Answer:**
import is used to access code from another module.

Example:

```python

```

---

```bash

```
