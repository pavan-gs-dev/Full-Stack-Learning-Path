# 📘 Dart Collections and Null Safety

<h1 align="center">Lists, Sets, Maps & Null Safety</h1>

Before learning Flutter Widgets, you must understand Dart Collections because almost every Flutter app uses them.

Examples:

* Product Lists
* User Data
* API Responses
* Shopping Cart
* Student Records
* Employee Information

---

# What are Collections?

Collections are used to store multiple values in a single variable.

Dart provides three main collection types:

1. List
2. Set
3. Map

Think of them as different ways to organize data.

---

# 1. List

## What is a List?

A List stores multiple values in a specific order.

A List allows:

✅ Duplicate values

✅ Ordered data

✅ Access by index

---

## Real Life Example

Shopping List:

```text
Milk
Bread
Eggs
Rice
```

All items are stored in order.

---

## Creating a List

```dart
void main() {
  List<String> fruits = [
    "Apple",
    "Mango",
    "Banana"
  ];

  print(fruits);
}
```

Output:

```text
[Apple, Mango, Banana]
```

---

# List Index

Every item has a position.

```text
Index      Value

0          Apple
1          Mango
2          Banana
```

---

## Accessing Items

```dart
void main() {
  List<String> fruits = [
    "Apple",
    "Mango",
    "Banana"
  ];

  print(fruits[0]);
}
```

Output:

```text
Apple
```

---

## Access Multiple Values

```dart
print(fruits[1]);
print(fruits[2]);
```

Output:

```text
Mango
Banana
```

---

# 2. Set

## What is a Set?

A Set stores unique values.

Duplicates are not allowed.

---

## Real Life Example

Student Roll Numbers

```text
101
102
103
```

Each roll number should appear only once.

---

## Creating a Set

```dart
void main() {
  Set<String> colors = {
    "Red",
    "Green",
    "Blue"
  };

  print(colors);
}
```

Output:

```text
{Red, Green, Blue}
```

---

## Duplicate Example

```dart
void main() {
  Set<String> colors = {
    "Red",
    "Blue",
    "Red"
  };

  print(colors);
}
```

Output:

```text
{Red, Blue}
```

Notice:

Second "Red" is automatically removed.

---

# When to Use Set?

Use Set when:

* Duplicate values are not allowed
* Unique IDs
* Unique Emails
* Unique Usernames

---

# 3. Map

## What is a Map?

A Map stores data as:

```text
Key : Value
```

Think of it like a dictionary.

---

## Real Life Example

Student Information

```text
Name : Pavan
Age : 22
City : Bangalore
```

---

## Creating a Map

```dart
void main() {
  Map<String, dynamic> student = {
    "name": "Pavan",
    "age": 22,
    "city": "Bangalore"
  };

  print(student);
}
```

Output:

```text
{name: Pavan, age: 22, city: Bangalore}
```

---

## Accessing Map Values

```dart
print(student["name"]);
```

Output:

```text
Pavan
```

---

## More Examples

```dart
print(student["age"]);
print(student["city"]);
```

Output:

```text
22
Bangalore
```

---

# Collection Methods

Collections provide built-in methods.

These methods help us manipulate data.

---

# add()

Adds a new item.

```dart
void main() {
  List<String> fruits = ["Apple"];

  fruits.add("Mango");

  print(fruits);
}
```

Output:

```text
[Apple, Mango]
```

---

# remove()

Removes an item.

```dart
void main() {
  List<String> fruits = [
    "Apple",
    "Mango"
  ];

  fruits.remove("Apple");

  print(fruits);
}
```

Output:

```text
[Mango]
```

---

# contains()

Checks whether an item exists.

```dart
void main() {
  List<String> fruits = [
    "Apple",
    "Mango"
  ];

  print(
    fruits.contains("Apple")
  );
}
```

Output:

```text
true
```

---

# length

Returns total number of items.

```dart
void main() {
  List<String> fruits = [
    "Apple",
    "Mango",
    "Banana"
  ];

  print(fruits.length);
}
```

Output:

```text
3
```

---

# List Looping

Loops are commonly used with Lists.

---

# for Loop with List

```dart
void main() {
  List<String> fruits = [
    "Apple",
    "Mango",
    "Banana"
  ];

  for(int i = 0;
      i < fruits.length;
      i++) {

    print(fruits[i]);
  }
}
```

Output:

```text
Apple
Mango
Banana
```

---

# for-in Loop

A cleaner way to loop through Lists.

---

## Syntax

```dart
for(variable in collection)
```

---

## Example

```dart
void main() {
  List<String> fruits = [
    "Apple",
    "Mango",
    "Banana"
  ];

  for(String fruit in fruits) {
    print(fruit);
  }
}
```

Output:

```text
Apple
Mango
Banana
```

---

## Why for-in?

Instead of:

```dart
fruits[0]
fruits[1]
fruits[2]
```

Dart automatically gives each item.

Cleaner and easier.

---

# forEach()

Another way to loop.

---

## Example

```dart
void main() {
  List<String> fruits = [
    "Apple",
    "Mango",
    "Banana"
  ];

  fruits.forEach((fruit) {
    print(fruit);
  });
}
```

Output:

```text
Apple
Mango
Banana
```

---

# Comparison

## for Loop

Best when index is needed.

```dart
for(int i = 0; i < list.length; i++)
```

---

## for-in

Best when only values are needed.

```dart
for(var item in list)
```

---

## forEach

Modern style.

```dart
list.forEach(...)
```

# Null Safety

## What is Null?

Null means:

```text
No Value
```

Example:

```dart
String? name = null;
```

The variable exists but contains no value.

---

# Why Null Safety?

Before Null Safety:

```dart
String name;
```

If name was empty:

```dart
name.length
```

Application could crash.

Null Safety prevents this problem.

---

# Nullable Variable (?)

Question mark means:

```text
This variable can contain null.
```

---

## Example

```dart
String? name = null;

print(name);
```

Output:

```text
null
```

---

## Another Example

```dart
String? name = "Pavan";

print(name);
```

Output:

```text
Pavan
```

---

# Non-Null Assertion (!)

Exclamation mark tells Dart:

```text
Trust me, value is not null.
```

---

## Example

```dart
String? name = "Pavan";

print(name!.length);
```

Output:

```text
5
```

---

## Danger

```dart
String? name = null;

print(name!.length);
```

Output:

```text
Error
```

Because value is actually null.

Use carefully.

---

# Null Coalescing Operator (??)

Provides a default value.

---

## Example

```dart
void main() {
  String? name = null;

  print(name ?? "Guest");
}
```

Output:

```text
Guest
```

---

## Another Example

```dart
String? name = "Pavan";

print(name ?? "Guest");
```

Output:

```text
Pavan
```

---

# Null Assignment Operator (??=)

Assign value only if variable is null.

---

## Example

```dart
void main() {
  String? name;

  name ??= "Guest";

  print(name);
}
```

Output:

```text
Guest
```

---

## Example 2

```dart
void main() {
  String? name = "Pavan";

  name ??= "Guest";

  print(name);
}
```

Output:

```text
Pavan
```

Because value already exists.

---

# Real World Flutter Example

```dart
String? username;
```

Show:

```dart
Text(
  username ?? "Guest User"
)
```

Output:

```text
Guest User
```

If username is null.

Otherwise actual username appears.
