# 1. What is Flutter?

Flutter is Google's UI toolkit used to build:

* Android Apps
* iOS Apps
* Web Apps
* Desktop Apps

using a **single codebase**.

Flutter uses the **Dart** programming language.

# First Flutter Program

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
```

# Why StatelessWidget?

```dart

```

Stateless means:


Example:


The text never changes.

So StatelessWidget is enough.

# Flutter Architecture

High-Level Architecture:

```text
Application
     |
 Widgets
     |
 Elements
     |
 Render Objects
     |
 Skia Engine
     |
 Device Screen
```

---

## Layer 1: Widgets

Developer writes:

```dart
Text("Hello")
```

Widgets are configurations.

They describe UI.

---

## Layer 2: Element Tree

Flutter creates:

```text
Element Objects
```

Elements connect widgets to render objects.

They manage lifecycle.

---

## Layer 3: Render Objects

Responsible for:

* Layout
* Size
* Position
* Painting

Example:

```text
Text Widget
      ↓
RenderParagraph
```

---

## Layer 4: Engine

Flutter Engine written in C++.

Handles:

* Rendering
* Animations
* GPU Calls
* Platform Communication

---

## Layer 5: Skia

Graphics Engine.

Draws pixels directly.

```text
Flutter
   ↓
Skia
   ↓
Screen
```

This is why Flutter is extremely fast.

# StatefulWidget

State can change.

Example:

```dart
class CounterPage extends StatefulWidget {
  @override
  State<CounterPage> createState() {
    return _CounterPageState();
  }
}

class _CounterPageState extends State<CounterPage> {

  int count = 0;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Text("$count"),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: () {
          setState(() {
            count++;
          });
        },
        child: Icon(Icons.add),
      ),
    );
  }
}
```

# What is setState()?

```dart
setState(() {
  count++;
});
```

Tells Flutter:

```text

Rebuild UI.
```

Without:

```dart
count++;
```

UI won't update.

# Hot Reload

Shortcut:

```text
r
```

or

```text
Ctrl + S
```

Behavior:

```text
Keeps App State
Rebuilds Widgets
Fast
```

Example:

Change:

```dart
Text("Hello")
```

to:

```dart
Text("Flutter")
```

Instantly updates.

---

# Hot Restart

Shortcut:

```text
Shift + R
```

Behavior:

```text
Destroys State
Runs main() again
```

Everything starts from zero.

---

# Hot Reload vs Hot Restart

| Feature          | Hot Reload | Hot Restart |
| ---------------- | ---------- | ----------- |
| Speed            | Fast       | Slower      |
| Keeps State      | Yes        | No          |
| Runs main()      | No         | Yes         |
| Rebuild UI       | Yes        | Yes         |
| Clears Variables | No         | Yes         |

✅ Everything in Flutter is a Widget

✅ `runApp()` starts app

✅ `Scaffold` creates screen structure
