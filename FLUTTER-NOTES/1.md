# App Development

## 1. Native App Development

Native app development means building apps specifically for a single platform.

### Android
- Java
- Kotlin

### iOS
- Objective-C
- Swift

### Advantages
- Better performance
- Full access to device features
- Best user experience

---

## 2. Hybrid / Cross-Platform App Development

Hybrid or cross-platform development allows developers to write a single codebase and deploy it on multiple platforms.

### Frameworks

#### Flutter
- One codebase
- Supports Android and iOS
- Uses Dart programming language

#### React Native
- One codebase
- Supports Android and iOS
- Uses JavaScript / TypeScript

### Advantages
- Faster development
- Lower maintenance cost
- Shared code across platforms

---

## Comparison

| Feature | Native Apps | Cross-Platform Apps |
|----------|------------|---------------------|
| Codebase | Separate | Single |
| Performance | Excellent | Good |
| Development Time | More | Less |
| Cost | Higher | Lower |
| Android Support | Yes | Yes |
| iOS Support | Yes | Yes |

---

## Technologies Overview

### Native Development
- Android → Java, Kotlin
- iOS → Objective-C, Swift

### Cross-Platform Development
- Flutter → Android & iOS
- React Native → Android & iOS

---

## Summary

- Use **Native Development** when maximum performance and platform-specific features are required.
- Use **Flutter** or **React Native** when you want to build Android and iOS applications using a single codebase.

# Flutter Math Operations

## Introduction

Flutter uses the Dart programming language. Dart provides basic mathematical operators to perform calculations.

---

# Addition (+)

Adds two numbers.

```dart
void main() {
  int a = 10;
  int b = 5;

  print(a + b);
}
```

Output:

```text
15
```

---

# Subtraction (-)

Subtracts one number from another.

```dart
void main() {
  int a = 10;
  int b = 5;

  print(a - b);
}
```

Output:

```text
5
```

---

# Multiplication (*)

Multiplies two numbers.

```dart
void main() {
  int a = 10;
  int b = 5;

  print(a * b);
}
```

Output:

```text
50
```

---

# Division (/)

Divides numbers and returns a decimal value.

```dart
void main() {
  int a = 10;
  int b = 5;

  print(a / b);
}
```

Output:

```text
2.0
```

---

# Integer Division (~/)

Returns only the whole number.

```dart
void main() {
  print(10 ~/ 3);
}
```

Output:

```text
3
```

---

# Modulus (%)

Returns the remainder.

```dart
void main() {
  print(10 % 3);
}
```

Output:

```text
1
```

---

# Increment (++)

Increases value by 1.

```dart
void main() {
  int x = 5;

  x++;

  print(x);
}
```

Output:

```text
6
```

---

# Decrement (--)

Decreases value by 1.

```dart
void main() {
  int x = 5;

  x--;

  print(x);
}
```

Output:

```text
4
```

---

# Complete Example

```dart
void main() {
  int a = 20;
  int b = 5;

  print("Addition: ${a + b}");
  print("Subtraction: ${a - b}");
  print("Multiplication: ${a * b}");
  print("Division: ${a / b}");
  print("Integer Division: ${a ~/ b}");
  print("Modulus: ${a % b}");
}
```

Output:

```text
Addition: 25
Subtraction: 15
Multiplication: 100
Division: 4.0
Integer Division: 4
Modulus: 0
```
