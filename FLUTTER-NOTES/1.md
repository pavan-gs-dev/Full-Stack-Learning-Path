# App Development

## 1. Native App Development

Native app development means building apps specifically for a single platform.

### Android
- Java
- Kotlin

### iOS
- Objective-C
- Swift

### Advantages
- Better performance
- Full access to device features
- Best user experience

---

## 2. Hybrid / Cross-Platform App Development

Hybrid or cross-platform development allows developers to write a single codebase and deploy it on multiple platforms.

### Frameworks

#### Flutter
- One codebase
- Supports Android and iOS
- Uses Dart programming language

#### React Native
- One codebase
- Supports Android and iOS
- Uses JavaScript / TypeScript

### Advantages
- Faster development
- Lower maintenance cost
- Shared code across platforms

---

## Comparison

| Feature | Native Apps | Cross-Platform Apps |
|----------|------------|---------------------|
| Codebase | Separate | Single |
| Performance | Excellent | Good |
| Development Time | More | Less |
| Cost | Higher | Lower |
| Android Support | Yes | Yes |
| iOS Support | Yes | Yes |

---

## Technologies Overview

### Native Development
- Android → Java, Kotlin
- iOS → Objective-C, Swift

### Cross-Platform Development
- Flutter → Android & iOS
- React Native → Android & iOS

---

## Summary

- Use **Native Development** when maximum performance and platform-specific features are required.
- Use **Flutter** or **React Native** when you want to build Android and iOS applications using a single codebase.

# **Flutter**

## What is Flutter?
### Flutter is an open-source UI framework developed by Google for building applications from a single codebase.

## What Language Does Flutter Use?
### Flutter uses: **Dart** 
### Dart was also developed by Google.

```dart
void main() {
  print("Hello Flutter");
}
```
## Flutter App Structure
`A basic Flutter app:`

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        body: Center(
          child: Text("Hello Flutter"),
        ),
      ),
    );
  }
}
```

# 🚀 What is DartPad?

**DartPad** is a free online editor where you can write and run **Dart** and **Flutter** code directly in your web browser.

You don't need to install:

* Flutter
* Dart SDK
* Android Studio

Just open DartPad and start coding instantly.

---

## Simple Dart Example

```dart
void main() {
  print("Hello World");
}
```

### Output

```text
Hello World
```

---

## Simple Variable Example

```dart
void main() {
  String name = "Pavan";
  int age = 22;

  print(name);
  print(age);
}
```

### Output

```text
Pavan
22
```
