# Flutter (Dart) Variables, var & const & final keyword & dynamic keyword 

## What is a Variable?

A variable is a container used to store data in memory.

Think of a variable as a labeled box where you keep information that can be used later in your program.

Example:

```dart
String studentName = "Rahul";
```

Here:

* `String` → Data type
* `studentName` → Variable name
* `"Rahul"` → Stored value

---

# Why Do We Use Variables?

Variables help us:

* Store data
* Reuse data
* Update data
* Perform calculations
* Display information to users

Example:

```dart
void main() {
  String userName = "Rahul";

  print(userName);
}
```

Output:

```text
Rahul
```

---

# Rules for Naming Variables

## Good Variable Names

Use meaningful names.

```dart
String studentName = "Rahul";
int studentAge = 21;
double productPrice = 999.99;
```

---

## Bad Variable Names

Avoid unclear names.

```dart
String x = "Rahul";
int a = 21;
double p = 999.99;
```

These names make code difficult to understand.

---

# Common Data Types

## String

Stores text.

```dart
String cityName = "Bangalore";
```

---

## int

Stores whole numbers.

```dart
int age = 22;
```

---

## double

Stores decimal numbers.

```dart
double height = 5.9;
```

---

## bool

Stores true or false.

```dart
bool isLoggedIn = true;
```

---

# Example Using Multiple Variables

```dart
void main() {
  String studentName = "Rahul";
  int age = 22;
  double height = 5.8;
  bool isStudent = true;

  print(studentName);
  print(age);
  print(height);
  print(isStudent);
}
```

Output:

```text
Rahul
22
5.8
true
```

# var Keyword

Dart can automatically determine the data type.

```dart
void main() {
  var studentName = "Rahul";

  print(studentName);
}
```

Dart understands that `studentName` is a String.

---

# final Keyword

## Definition

`final` means the value can be assigned only once.

After assigning a value, it cannot be changed.

---

## Example

```dart
void main() {
  final String collegeName = "ABC College";

  print(collegeName);
}
```

Output:

```text
ABC College
```

---

## Error Example

```dart
final String collegeName = "ABC College";

collegeName = "XYZ College";
```

Output:

```text
Error
```

Because a final variable cannot be reassigned.

---

## When to Use final?

Use `final` when:

* Value is known only at runtime
* Value should not change after assignment

Examples:

```dart
final currentDate = DateTime.now();
final userId = 1001;
```

---

# const Keyword

## Definition

`const` creates a compile-time constant.

The value must be known before the program runs.

---

## Example

```dart
const double pi = 3.14159;

void main() {
  print(pi);
}
```

Output:

```text
3.14159
```

---

## Error Example

```dart
const currentDate = DateTime.now();
```

Output:

```text
Error
```

Because `DateTime.now()` is determined while the program is running.

---

# Difference Between final and const

| Feature                     | final | const |
| --------------------------- | ----- | ----- |
| Value assigned once         | ✅     | ✅     |
| Can change later            | ❌     | ❌     |
| Runtime value allowed       | ✅     | ❌     |
| Compile-time value required | ❌     | ✅     |

---

## final Example

```dart
final currentTime = DateTime.now();
```

Valid because value is generated during runtime.

---

## const Example

```dart
const countryName = "India";
```

Valid because value is known before execution.

---

# dynamic Keyword

## Definition

`dynamic` means the variable can store any type of value.

The type can change during execution.

---

## Example

```dart
void main() {
  dynamic data = "Rahul";

  print(data);

  data = 100;

  print(data);

  data = true;

  print(data);
}
```

Output:

```text
Rahul
100
true
```

---

# Why Use dynamic?

Useful when:

* Data type is unknown
* API responses can vary
* Working with JSON data

Example:

```dart
dynamic apiResponse;
```

---

# Problem with dynamic

Since Dart cannot check the type beforehand, errors may occur at runtime.

Example:

```dart
dynamic value = "Rahul";

print(value.toUpperCase());
```

Works.

But:

```dart
dynamic value = 100;

print(value.toUpperCase());
```

Runtime Error.

Because integers do not have `toUpperCase()`.

---

# Recommended Practice

Prefer:

```dart
String studentName = "Rahul";
int age = 22;
double salary = 25000.0;
```

Use:

```dart
dynamic
```

only when necessary.

# Complete Example

```dart
void main() {
  String studentName = "Rahul";
  int age = 22;

  final String collegeName = "ABC College";

  const String countryName = "India";

  dynamic data = "Flutter";

  print(studentName);
  print(age);
  print(collegeName);
  print(countryName);
  print(data);

  data = 100;

  print(data);
}
```

Output:

```text
Rahul
22
ABC College
India
Flutter
100
```

# Summary

| Keyword | Purpose                         |
| ------- | ------------------------------- |
| String  | Store text                      |
| int     | Store whole numbers             |
| double  | Store decimal numbers           |
| bool    | Store true or false             |
| var     | Dart automatically detects type |
| final   | Value assigned once at runtime  |
| const   | Compile-time constant           |
| dynamic | Type can change during runtime  |

## Best Practice

* Use meaningful variable names.
* Prefer specific types (`String`, `int`, `double`).
* Use `final` for values that should not change.
* Use `const` for fixed constants.
* Use `dynamic` only when the data type is unknown.
