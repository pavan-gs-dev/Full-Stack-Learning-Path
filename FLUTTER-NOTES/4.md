# 📘 Dart Control Flow Statements

<h1 align="center">Control Flow in Dart</h1>

Control Flow determines how a program makes decisions and executes different blocks of code.

Think of Control Flow like traffic signals:

* Green Light → Go
* Red Light → Stop
* Yellow Light → Wait

Similarly, Dart uses conditions to decide what code should run.

---

# 1. if Statement

## What is an if Statement?

The `if` statement executes a block of code only when a condition is true.

### Syntax

```dart
if (condition) {
  // code
}
```

---

## Simple Example

```dart
void main() {
  int age = 20;

  if (age >= 18) {
    print("You can vote");
  }
}
```

### Output

```text
You can vote
```

### Explanation

* age = 20
* 20 >= 18 is true
* Code inside if executes

---

## Real Life Example

```dart
void main() {
  double balance = 5000;

  if (balance >= 1000) {
    print("Withdrawal Allowed");
  }
}
```

Output:

```text
Withdrawal Allowed
```

---

# 2. if-else Statement

## What is if-else?

If condition is true, run one block.

Otherwise run another block.

### Syntax

```dart
if (condition) {
  // true block
} else {
  // false block
}
```

---

## Example

```dart
void main() {
  int age = 15;

  if (age >= 18) {
    print("Eligible to Vote");
  } else {
    print("Not Eligible");
  }
}
```

### Output

```text
Not Eligible
```

---

## How it Works

Condition:

```dart
age >= 18
```

Result:

```dart
15 >= 18
```

Answer:

```dart
false
```

Therefore else block runs.

---

# 3. if-else-if Ladder

Used when multiple conditions need to be checked.

---

## Example

```dart
void main() {
  int marks = 85;

  if (marks >= 90) {
    print("Grade A");
  } else if (marks >= 75) {
    print("Grade B");
  } else if (marks >= 50) {
    print("Grade C");
  } else {
    print("Fail");
  }
}
```

### Output

```text
Grade B
```

---

## Real World Example

Student Results:

| Marks    | Grade |
| -------- | ----- |
| 90+      | A     |
| 75+      | B     |
| 50+      | C     |
| Below 50 | Fail  |

# 4. Conditional Operator (Ternary Operator)

## What is Conditional Operator?

A short form of if-else.

Instead of writing:

```dart
if (condition) {
  value1
} else {
  value2
}
```

You can write:

```dart
condition ? value1 : value2;
```

---

## Syntax

```dart
condition ? trueValue : falseValue;
```

---

## Example

```dart
void main() {
  int age = 20;

  String result =
      age >= 18 ? "Adult" : "Minor";

  print(result);
}
```

### Output

```text
Adult
```

---

## Without Ternary

```dart
if (age >= 18) {
  result = "Adult";
} else {
  result = "Minor";
}
```

---

## With Ternary

```dart
result = age >= 18
    ? "Adult"
    : "Minor";
```

Cleaner and shorter.

# 5. Switch Statement

## What is Switch?

Switch is used when you have many possible values.

Instead of using many if-else statements.

---

## Syntax

```dart
switch (value) {
  case something:
    code;
    break;

  default:
    code;
}
```

---

## Example

```dart
void main() {
  int day = 3;

  switch (day) {
    case 1:
      print("Monday");
      break;

    case 2:
      print("Tuesday");
      break;

    case 3:
      print("Wednesday");
      break;

    default:
      print("Invalid Day");
  }
}
```

### Output

```text
Wednesday
```

---

## Real Life Example

```dart
void main() {
  String role = "admin";

  switch (role) {
    case "admin":
      print("Full Access");
      break;

    case "user":
      print("Limited Access");
      break;

    default:
      print("Guest Access");
  }
}
```

---

# 6. Dart 3 Switch Expression

Dart 3 introduced a more modern and cleaner switch syntax.

---

## Old Switch

```dart
switch(day) {
  case 1:
    print("Monday");
    break;
}
```

---

## Dart 3 Switch Expression

```dart
void main() {
  int day = 2;

  String result = switch (day) {
    1 => "Monday",
    2 => "Tuesday",
    3 => "Wednesday",
    _ => "Invalid Day"
  };

  print(result);
}
```

### Output

```text
Tuesday
```

---

## Why Dart 3 Switch is Better?

✅ Less code

✅ Easy to read

✅ Faster to write

✅ Modern Dart style

# 7. Enums

## What is an Enum?

Enum stands for Enumeration.

An Enum is used to create a fixed set of values.

Example:

Days of Week:

* Monday
* Tuesday
* Wednesday
* Thursday
* Friday

Only these values are allowed.

---

## Why Use Enum?

Without Enum:

```dart
String day = "Monday";
```

Problem:

```dart
String day = "Mondya";
```

Typing mistake.

Program may fail.

---

With Enum:

```dart
enum Day {
  monday,
  tuesday,
  wednesday
}
```

Only valid values can be used.

---

# Creating an Enum

```dart
enum Day {
  monday,
  tuesday,
  wednesday,
  thursday,
  friday
}
```

---

# Using Enum

```dart
void main() {
  Day today = Day.monday;

  print(today);
}
```

### Output

```text
Day.monday
```

---

# Enum with Switch

```dart
enum Day {
  monday,
  tuesday,
  wednesday
}

void main() {
  Day today = Day.tuesday;

  switch (today) {
    case Day.monday:
      print("Start of Week");
      break;

    case Day.tuesday:
      print("Working Day");
      break;

    case Day.wednesday:
      print("Mid Week");
      break;
  }
}
```

### Output

```text
Working Day
```
