# 📘 Dart Type Conversion, Strings, and Boolean Operations

---

# 1. Type Conversion (Type Casting)

## What is Type Conversion?

Type Conversion is the process of converting one data type into another data type.

For example:

* String → int
* String → double
* int → String
* double → int

Type conversion is very common when working with user input, APIs, databases, and files because data often comes as text (String).

---

## Why Do We Need Type Conversion?

Suppose the user enters a number:

```dart
String age = "22";
```

The value looks like a number, but Dart treats it as a String.

You cannot perform mathematical operations on a String.

❌ Wrong:

```dart
String age = "22";

print(age + 5);
```

Output:

```text
Error
```

You must convert it to an integer first.

✅ Correct:

```dart
String age = "22";

int newAge = int.parse(age);

print(newAge + 5);
```

Output:

```text
27
```

---

## String to int

```dart
void main() {
  String number = "100";

  int value = int.parse(number);

  print(value);
}
```

Output:

```text
100
```

---

## String to double

```dart
void main() {
  String price = "99.99";

  double value = double.parse(price);

  print(value);
}
```

Output:

```text
99.99
```

---

## int to String

```dart
void main() {
  int age = 22;

  String text = age.toString();

  print(text);
}
```

Output:

```text
22
```

---

## double to String

```dart
void main() {
  double marks = 95.5;

  String result = marks.toString();

  print(result);
}
```

Output:

```text
95.5
```

---

## double to int

When converting double to int, decimal values are removed.

```dart
void main() {
  double value = 25.99;

  int number = value.toInt();

  print(number);
}
```

Output:

```text
25
```

# 2. String in Dart

## What is a String?

A String is a collection of characters enclosed inside quotes.

Strings are used to store text data.

Examples:

```dart
String name = "Pavan";
String city = "Bangalore";
String company = "Google";
```

---

## Creating Strings

Using Double Quotes

```dart
String name = "Pavan";
```

Using Single Quotes

```dart
String city = 'Bangalore';
```

Both are valid.

---

## String Properties

```dart
void main() {
  String name = "Flutter";

  print(name.length);
}
```

Output:

```text
7
```

---

# 3. String Concatenation

## What is String Concatenation?

Joining two or more strings together is called String Concatenation.

---

## Using + Operator

```dart
void main() {
  String firstName = "Pavan";
  String lastName = "Kumar";

  String fullName = firstName + " " + lastName;

  print(fullName);
}
```

Output:

```text
Pavan Kumar
```

---

## Multiple Strings

```dart
void main() {
  String a = "Hello";
  String b = "World";

  print(a + " " + b);
}
```

Output:

```text
Hello World
```

---

# 4. String Interpolation

## What is String Interpolation?

String interpolation allows variables to be inserted directly inside a String.

Instead of using + operator repeatedly, Dart provides:

```dart
$variable
```

and

```dart
${expression}
```

---

## Using Variables

```dart
void main() {
  String name = "Pavan";

  print("My name is $name");
}
```

Output:

```text
My name is Pavan
```

---

## Using Expressions

```dart
void main() {
  int a = 10;
  int b = 20;

  print("Sum = ${a + b}");
}
```

Output:

```text
Sum = 30
```

---

## Why Interpolation is Better?

❌ Old Way

```dart
print("Name: " + name);
```

✅ Better Way

```dart
print("Name: $name");
```

Cleaner and easier to read.

# 5. Multiline String

## What is a Multiline String?

A multiline string allows text to span multiple lines.

Dart uses triple quotes.

---

## Triple Double Quotes

```dart
void main() {
  String text = """
Hello
Welcome to Dart
Programming Language
""";

  print(text);
}
```

Output:

```text
Hello
Welcome to Dart
Programming Language
```

---

## Triple Single Quotes

```dart
void main() {
  String text = '''
Flutter
Android
iOS
''';

  print(text);
}
```

Output:

```text
Flutter
Android
iOS
```

---

## Real World Example

```dart
void main() {
  String address = """
Pavan Kumar
Ramanagara
Karnataka
India
""";

  print(address);
}
```

# 6. Boolean (bool)

## What is bool?

A bool represents only two values:

```dart
true
false
```

Used for decision making.

Examples:

```dart
bool isLoggedIn = true;

bool isAdmin = false;
```

---

## Example

```dart
void main() {
  bool isFlutterAwesome = true;

  print(isFlutterAwesome);
}
```

Output:

```text
true
```

# 7. Relational Operators

## What are Relational Operators?

Relational operators compare two values.

The result is always:

```dart
true
```

or

```dart
false
```

---

## List of Relational Operators

| Operator | Meaning                  |
| -------- | ------------------------ |
| ==       | Equal To                 |
| !=       | Not Equal To             |
| >        | Greater Than             |
| <        | Less Than                |
| >=       | Greater Than or Equal To |
| <=       | Less Than or Equal To    |

---

## Equal To (==)

```dart
void main() {
  print(10 == 10);
}
```

Output:

```text
true
```

---

## Not Equal To (!=)

```dart
void main() {
  print(10 != 5);
}
```

Output:

```text
true
```

---

## Greater Than (>)

```dart
void main() {
  print(20 > 10);
}
```

Output:

```text
true
```

---

## Less Than (<)

```dart
void main() {
  print(5 < 10);
}
```

Output:

```text
true
```

---

## Greater Than or Equal To (>=)

```dart
void main() {
  print(20 >= 20);
}
```

Output:

```text
true
```

---

## Less Than or Equal To (<=)

```dart
void main() {
  print(10 <= 20);
}
```

Output:

```text
true
```

---

# Real-World Example Using bool and Relational Operators

```dart
void main() {
  int age = 22;

  bool canVote = age >= 18;

  print(canVote);
}
```

Output:

```text
true
```

Explanation:

* age is 22
* 22 is greater than 18
* Therefore result is true
