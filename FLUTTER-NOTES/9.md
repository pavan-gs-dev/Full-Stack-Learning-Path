# dart:convert

Used for JSON operations.

Import:

```dart

```

---

# Convert JSON String to Map

```dart
import 'dart:convert';

void main() {

  String jsonString =
      '{"name":"Pavan"}';

  var data =
      jsonDecode(jsonString);

  print(data["name"]);

}
```

Output:

```text
Pavan
```

---

# Convert Map to JSON

```dart
import 'dart:convert';

void main() {

  Map<String, dynamic> user = {
    "name": "Pavan"
  };

  print(jsonEncode(user));

}
```

Output:

```text
{"name":"Pavan"}
```

A Package is a collection of libraries distributed for reuse.

Think of:


# Hiding Specific Members

```dart
import 'dart:math' hide max;
```

Now:

```dart
max()
```

cannot be used.

---

# Showing Specific Members

```dart
import 'dart:math' show min;
```

Only:

```dart
min()
```

is available.

---

# Alias Import

Useful when library names conflict.

```dart
import 'dart:math' as math;
```

Usage:

```dart
print(math.max(10, 20));
```

Output:

```text
20
```
