# 📘 Dart Loops and Functions

<h1 align="center">Loops & Functions in Dart</h1>

Loops help us repeat code multiple times.

Functions help us organize and reuse code.

Without loops and functions, programming would become repetitive and difficult to manage.

---

# What is a Loop?

A loop is used to execute the same block of code multiple times.

## Real Life Example

Imagine you want to print:

```text
Welcome
Welcome
Welcome
Welcome
Welcome
```

Without a loop:

```dart
print("Welcome");
print("Welcome");
print("Welcome");
print("Welcome");
print("Welcome");
```

This is repetitive.

Using a loop:

```dart
for(int i = 1; i <= 5; i++) {
  print("Welcome");
}
```

Same output with less code.

---

# Why Use Loops?

Loops help:

* Reduce code repetition
* Save time
* Make code cleaner
* Process large amounts of data

---

# 1. For Loop

The for loop is the most commonly used loop in Dart.

It is used when we know how many times the loop should run.

---

## Syntax

```dart
for(initialization; condition; update) {
   // code
}
```

---

## Understanding Each Part

### Initialization

```dart
int i = 1;
```

Starting value.

---

### Condition

```dart
i <= 5
```

Loop runs while condition is true.

---

### Update

```dart
i++
```

Increase value after every iteration.

---

## Example 1

```dart
void main() {
  for(int i = 1; i <= 5; i++) {
    print(i);
  }
}
```

### Output

```text
1
2
3
4
5
```

---

## How It Works

### First Iteration

```dart
i = 1
```

Condition:

```dart
1 <= 5
```

True

Print:

```text
1
```

Increase:

```dart
i++
```

Now:

```dart
i = 2
```

Same process continues until:

```dart
i = 6
```

Condition becomes false.

Loop stops.

---

## Example 2

Print Even Numbers

```dart
void main() {
  for(int i = 2; i <= 10; i += 2) {
    print(i);
  }
}
```

Output:

```text
2
4
6
8
10
```

---

## Example 3

Multiplication Table

```dart
void main() {
  int number = 5;

  for(int i = 1; i <= 10; i++) {
    print("$number x $i = ${number * i}");
  }
}
```

---

# 2. While Loop

A while loop runs as long as the condition remains true.

---

## Syntax

```dart
while(condition) {
   // code
}
```

---

## Example

```dart
void main() {
  int i = 1;

  while(i <= 5) {
    print(i);
    i++;
  }
}
```

Output:

```text
1
2
3
4
5
```

---

## How While Works

Step 1

```dart
i = 1
```

Condition:

```dart
i <= 5
```

True

Print 1

Increase i

Repeat.

---

## Real Life Example

ATM Machine

Keep asking for PIN until correct PIN is entered.

```dart
while(pinIncorrect) {
   askForPin();
}
```

---

# Difference Between For and While

## For Loop

Use when number of repetitions is known.

Example:

```dart
for(int i = 1; i <= 10; i++)
```

---

## While Loop

Use when repetitions are unknown.

Example:

```dart
while(userLoggedIn)
```

---

# 3. Do While Loop

A do while loop executes the code at least once.

Even if the condition is false.

---

## Syntax

```dart
do {
   // code
}
while(condition);
```

---

## Example

```dart
void main() {
  int i = 1;

  do {
    print(i);
    i++;
  }
  while(i <= 5);
}
```

Output:

```text
1
2
3
4
5
```

---

## Special Example

```dart
void main() {
  int i = 10;

  do {
    print("Executed");
  }
  while(i < 5);
}
```

Output:

```text
Executed
```

---

## Why?

Because do block executes first.

Condition is checked later.

---

# Comparison

| Loop     | Condition Checked |
| -------- | ----------------- |
| for      | Before execution  |
| while    | Before execution  |
| do while | After execution   |

---

# 4. Nested Loop

A loop inside another loop is called a Nested Loop.

---

## Structure

```dart
for(...) {

   for(...) {

   }

}
```

---

## Example

```dart
void main() {
  for(int row = 1; row <= 3; row++) {

    for(int col = 1; col <= 3; col++) {
      print("Row $row Column $col");
    }

  }
}
```

Output:

```text
Row 1 Column 1
Row 1 Column 2
Row 1 Column 3
Row 2 Column 1
Row 2 Column 2
Row 2 Column 3
Row 3 Column 1
Row 3 Column 2
Row 3 Column 3
```

---

# Star Pattern Example

```dart
void main() {
  for(int i = 1; i <= 5; i++) {

    String stars = "";

    for(int j = 1; j <= i; j++) {
      stars += "*";
    }

    print(stars);
  }
}
```

Output:

```text
*
**
***
****
*****
```

# What is a Function?

A function is a block of code that performs a specific task.

Instead of writing the same code repeatedly, we place it inside a function and call it whenever needed.

---

## Real Life Example

Think of a function like a TV remote button.

Whenever you press:

```text
Power Button
```

The TV turns on.

Similarly:

```dart
greetUser();
```

Runs a predefined block of code.

---

# Function Creation

## Syntax

```dart
returnType functionName() {

}
```

---

## Simple Function

```dart
void greet() {
  print("Welcome to Dart");
}
```

Calling Function

```dart
void main() {
  greet();
}
```

Output:

```text
Welcome to Dart
```

---

# Why Functions?

Without Function

```dart
print("Welcome");
print("Welcome");
print("Welcome");
```

With Function

```dart
greet();
greet();
greet();
```

Cleaner code.

---

# Function With Parameters

Parameters allow us to pass values.

---

## Example

```dart
void greet(String name) {
  print("Hello $name");
}

void main() {
  greet("Pavan");
}
```

Output:

```text
Hello Pavan
```

---

# Function With Multiple Parameters

```dart
void student(String name, int age) {
  print("Name: $name");
  print("Age: $age");
}
```

Calling:

```dart
student("Pavan", 22);
```

Output:

```text
Name: Pavan
Age: 22
```

---

# Function Returning Value

Functions can return data.

---

## Example

```dart
int add(int a, int b) {
  return a + b;
}

void main() {
  int result = add(10, 20);

  print(result);
}
```

Output:

```text
30
```

---

# Function Types

## No Parameter, No Return

```dart
void greet() {
  print("Hello");
}
```

---

## Parameter, No Return

```dart
void greet(String name) {
  print(name);
}
```

---

## No Parameter, Return Value

```dart
int getAge() {
  return 22;
}
```

---

## Parameter and Return Value

```dart
int multiply(int a, int b) {
  return a * b;
}
```
